# C29 — GOVERNANCE / INVESTOR DASHBOARD REAL

## Objetivo
Construir un dashboard real de métricas del negocio y del sistema, sin teatro corporativo.

## Qué implementar
1. Métricas reales existentes:
   - ejecuciones
   - uso por plan
   - upgrades
   - errores
   - handoffs
   - installs/update readiness si aplica
2. Vista ejecutiva simple.
3. Fuente de datos real.

## Definition of Done
- dashboard con datos reales del sistema
- sin KPIs inventados
- útil para fundador/inversor interno

## Demo obligatoria
“Vista ejecutiva con métricas reales del producto”.

## No hacer
- board governance fake
- estados financieros inventados
