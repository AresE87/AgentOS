# AgentOS v2 — Roadmap R71-R80: Ecosystem de desarrollo y herramientas next-gen

**Fecha:** 29 de marzo de 2026
**Estado:** R1-R70 completas. v1.2 Enterprise publicada con calendar, email, DB, API, sandbox, teams.
**Objetivo:** Hacer que DESARROLLADORES quieran construir sobre AgentOS. IDE para agentes, visual workflow builder, testing framework, version control para playbooks, y una plataforma de desarrollo madura.

---

## Las 10 fases R71-R80

| Fase | Nombre | Qué agrega |
|------|--------|-----------|
| R71 | **Visual workflow builder** | Drag-and-drop editor para crear chains sin código |
| R72 | **Webhook actions** | El agente recibe webhooks de GitHub/Stripe/Jira y actúa automáticamente |
| R73 | **Custom LLM fine-tuning** | Fine-tune un modelo small con los datos del usuario (local) |
| R74 | **Agent testing framework** | Unit tests para playbooks y specialists (TDD para agentes) |
| R75 | **Version control para playbooks** | Git-like history, diff, branching de playbooks |
| R76 | **Agent analytics pro** | Funnel analysis, cohort retention, predictive cost modeling |
| R77 | **Embeddable agent** | Widget que otros sitios web pueden embeber para dar soporte AI |
| R78 | **CLI power mode** | Terminal completa dentro de AgentOS con AI autocomplete |
| R79 | **Extension API v2** | Plugins crean páginas completas en el dashboard |
| R80 | **v1.3 — Developer release** | Docs v3, certificación de plugins, partner program |

**Regla:** Demo grabable. Cada feature hace que un developer diga "necesito construir sobre esto".
