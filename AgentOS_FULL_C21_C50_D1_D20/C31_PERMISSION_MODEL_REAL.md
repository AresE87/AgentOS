# C31 — PERMISSION MODEL REAL

## Objetivo
Construir un modelo real de permisos para agentes, usuarios y acciones.

## Qué implementar
1. Permisos por capacidad.
2. Scope por usuario/tenant/agente.
3. Rechazo explícito cuando falta permiso.
4. Estado observable de permisos.

## Definition of Done
- cada acción sensible verifica permisos reales
