# FASE R58 — TEMPLATE ENGINE: Generar documentos desde templates

**Objetivo:** El usuario define templates (email, reporte, propuesta, factura) con placeholders, y el agente los llena con datos reales. "Generá el reporte mensual" → documento completo basado en template + datos de analytics.

---

## Tareas

### 1. Template format

```
templates/
├── monthly-report.md
├── email-followup.md
├── project-proposal.md
├── invoice.md
└── weekly-standup.md
```

```markdown
<!-- templates/monthly-report.md -->
# Monthly Report — {{month}} {{year}}

## Summary
{{ai:generate a 2-sentence summary of this month's activity based on the analytics data}}

## Key Metrics
- Tasks completed: {{analytics.tasks_completed}}
- Success rate: {{analytics.success_rate}}%
- Total cost: ${{analytics.total_cost}}
- Time saved: {{analytics.time_saved_hours}} hours

## Top Playbooks
{{for playbook in analytics.top_playbooks}}
- {{playbook.name}}: {{playbook.count}} executions ({{playbook.success_rate}}% success)
{{endfor}}

## Highlights
{{ai:list 3 notable achievements or patterns from this month}}

## Recommendations
{{ai:suggest 3 improvements based on the data}}

---
Generated by AgentOS on {{date}}
```
