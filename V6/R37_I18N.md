# FASE R37 — INTERNACIONALIZACIÓN: Multi-idioma

**Objetivo:** La UI de AgentOS está disponible en español, inglés, y portugués. El agente detecta el idioma del usuario y responde en el mismo idioma.

---

### 4. Playbooks multiidioma

```json
// metadata.json del playbook:
{
  "name": {"en": "System Monitor", "es": "Monitor de Sistema", "pt": "Monitor do Sistema"},
  "description": {"en": "Monitors...", "es": "Monitorea...", "pt": "Monitora..."}
}
```

### 5. Cambio de idioma en Settings

```
LANGUAGE
  [English ▾]  ← dropdown con: English, Español, Português
  
  Agent language: [Same as UI ▾]  ← o override: "Always respond in English"
```

## Demo

1. Cambiar idioma a Español → toda la UI se traduce
2. Enviar mensaje en español → agente responde en español
3. Cambiar a Portugués → UI + agente en portugués
4. Playbook names se muestran en el idioma seleccionado
