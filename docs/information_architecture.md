# Information Architecture

## Dashboard structure
- Core
  - Home
  - Chat
  - Playbooks
  - Board
- Operate
  - Operations
  - Mesh
  - Analytics
  - Triggers
- Build
  - Developer
  - Settings
- Readiness
  - Readiness
  - Feedback

## Principle
- Core is for using AgentOS.
- Operate is for running the platform.
- Build is for inspecting and extending the system.
- Readiness is for proving external-facing maturity.

## Result
- The dashboard no longer hides operator and readiness surfaces behind generic labels alone.
