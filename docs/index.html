AgentOS Documentation

The universal AI agent for your PC. Controls your screen, runs commands, and automates anything.

Installation

Download the installer from GitHub Releases and run it. Requires Windows 10+.

# Or build from source:
git clone https://github.com/yourusername/agentos
cd agentos
cargo tauri build

Quick Start

  1. Install AgentOS
  2. Open the app and go to Settings
  3. Add your Anthropic API key
  4. Type a task in the chat: Check my disk space

API Keys Setup

AgentOS supports multiple AI providers. Add at least one to get started:

  • Anthropic Claude — Best for complex tasks. Get key →
  • OpenAI GPT — Alternative provider
  • Ollama — Run local models for free

Chat Interface

The chat interface accepts natural language commands. Examples:

"Check my disk space"
"Open Notepad and type Hello World"
"Find all PDF files in Downloads"
"Search for the latest news about AI"

Vision Control

AgentOS can see and control your screen. Enable vision tasks with:

"Click the Start button"
"Take a screenshot and describe what you see"
"Open Calculator and compute 15% of 350"

Playbooks

Playbooks are saved sequences of steps. Record one by clicking the Record button in the Playbooks tab.

Agent Chains

Complex tasks are automatically decomposed into subtasks and routed to specialist agents. Use the Chains tab to view execution history.

Triggers

Triggers run playbooks automatically on a schedule (cron) or when conditions are met. Configure them in the Triggers tab.

Mesh Networking

Connect multiple PCs in a local mesh network. Each node can receive and execute tasks from any other node.

Public API

AgentOS exposes a local REST API on port 8080. Enable it in Settings → API.

curl -H "Authorization: Bearer aos_yourkey" \
  -d '{"text":"check disk space"}' \
  http://localhost:8080/v1/message

Webhooks

Register webhook URLs to receive task completion notifications. See API Reference for details.
