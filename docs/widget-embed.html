AgentOS Embeddable Widget

Embed your AgentOS assistant on any website with a single code snippet.

Quick Start

1. Generate your embed snippet

Use the AgentOS dashboard or API to generate a snippet. The minimal configuration requires an API key and agent URL.

<!-- AgentOS Embeddable Widget -->
<div id="agentos-widget"
     data-api-key="YOUR_API_KEY"
     data-agent-url="https://your-agent.example.com"
     data-theme="light"
     data-position="bottom-right"
     data-welcome="Hello! How can I help you today?">
</div>
<script>
(function() {
  var w = document.getElementById('agentos-widget');
  var cfg = {
    apiKey: w.getAttribute('data-api-key'),
    agentUrl: w.getAttribute('data-agent-url'),
    theme: w.getAttribute('data-theme') || 'light',
    position: w.getAttribute('data-position') || 'bottom-right',
    welcome: w.getAttribute('data-welcome') || 'Hello!'
  };
  var iframe = document.createElement('iframe');
  iframe.src = cfg.agentUrl + '/widget?api_key=' + encodeURIComponent(cfg.apiKey)
    + '&theme=' + cfg.theme + '&position=' + cfg.position
    + '&welcome=' + encodeURIComponent(cfg.welcome);
  iframe.style.cssText = 'border:none;width:380px;height:520px;position:fixed;'
    + 'bottom:20px;right:20px;z-index:99999;border-radius:12px;'
    + 'box-shadow:0 4px 24px rgba(0,0,0,0.15);';
  w.appendChild(iframe);
})();
</script>
Configuration

2. Widget Options

AttributeRequiredDefaultDescription
data-api-keyYes-Your AgentOS API key
data-agent-urlYes-URL of your AgentOS instance
data-personaNodefaultAgent persona to use
data-themeNolight"light" or "dark"
data-positionNobottom-right"bottom-right", "bottom-left", "top-right", "top-left"
data-welcomeNoHello!Welcome message displayed on open
Iframe

3. Direct Iframe Embed

You can also embed the widget directly as an iframe:

<iframe
  src="https://your-agent.example.com/widget?api_key=YOUR_KEY&theme=dark"
  style="border:none;width:380px;height:520px;"
></iframe>
