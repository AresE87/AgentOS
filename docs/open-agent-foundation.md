# Open Agent Foundation Charter

**Version:** 1.0
**Established:** 2026-03-29

## Mission

The Open Agent Foundation exists to promote open standards, interoperability, and ethical development of autonomous AI agent systems. We believe AI agents should be transparent, user-controlled, and built on open protocols.

## Principles

1. **User Sovereignty** -- Users own their data and control their agents. No vendor lock-in.
2. **Interoperability** -- Agents from different vendors must communicate via open protocols (AAP).
3. **Transparency** -- Agent decision-making should be explainable and auditable.
4. **Safety First** -- All agents must implement kill switches, rate limits, and approval workflows.
5. **Privacy by Design** -- Data minimization, encryption at rest and in transit, GDPR compliance.
6. **Open Source Core** -- Core protocols and reference implementations remain open source.

## Governance

### Board of Directors
- 5 seats: 2 founding members, 2 elected community representatives, 1 independent advisor
- Terms: 2 years, staggered elections
- Decisions by simple majority; protocol changes require 4/5 supermajority

### Technical Steering Committee (TSC)
- Responsible for protocol specifications (AAP), reference implementations, and certification
- Open membership for active contributors
- RFC process for all specification changes

### Working Groups
- **Protocol WG** -- AAP specification evolution
- **Safety WG** -- Agent safety standards and testing frameworks
- **Interop WG** -- Cross-platform compatibility testing
- **Ethics WG** -- Responsible AI agent guidelines

## Membership

### Tiers
- **Community** -- Free, open to all individuals. Voting rights in community elections.
- **Contributor** -- For active code/spec contributors. TSC eligibility.
- **Corporate** -- For organizations adopting AAP. Advisory board participation.
- **Founding** -- Charter members with permanent board representation.

## Intellectual Property

- All specifications released under CC BY 4.0
- Reference implementations under Apache 2.0
- Patents: RAND-Z (Reasonable and Non-Discriminatory, Zero royalty) commitment required for all contributors

## Certification Program

Agents and platforms may apply for OAF certification:
- **AAP Compliant** -- Passes interoperability test suite
- **Safety Certified** -- Implements all required safety controls
- **Privacy Certified** -- Passes privacy audit checklist

## Funding

- Corporate membership dues
- Grant funding for research and development
- Conference and certification revenue
- No venture capital; foundation remains independent
