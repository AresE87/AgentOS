# AgentOS — Press Kit

## Nombre del producto

**AgentOS**

## Tagline

> "Tu fuerza de trabajo digital, corriendo localmente y bajo control."

## Descripcion

AgentOS es un sistema operativo de agentes de IA que corre en tu escritorio. Coordina equipos de agentes autonomos que ejecutan tareas reales: controlan tu pantalla, leen emails, gestionan tu agenda, generan contenido para redes sociales, y automatizan trabajo operativo complejo. Todo corre localmente, con tus datos bajo tu control. Compatible con modelos cloud (Anthropic, OpenAI, Google) y modelos locales gratuitos via Ollama.

## Features principales

- **Coordinator Mode:** Misiones multi-agente con DAG visual, subtareas paralelas, y agentes especializados que trabajan en equipo.
- **Training Studio + Marketplace:** Graba automatizaciones como "trainings" y vendilas en el marketplace integrado. Monetiza tu conocimiento operativo.
- **Marketing Engine:** Genera contenido para Twitter, LinkedIn, Reddit y Hacker News con un clic. Calendario editorial, campanas, y monitoreo de menciones.
- **Ejecucion local-first:** Tus datos nunca salen de tu maquina. Vault encriptado para API keys. Funciona offline con modelos locales.
- **Extensible:** +100 herramientas integradas, sistema de plugins, webhooks, API REST publica, y conectores para Calendar, Gmail, bases de datos, y mas.
