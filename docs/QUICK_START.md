# AgentOS — Quick Start

Tres pasos para empezar a usar tu fuerza de trabajo digital.

---

## 1. Instalar

Descarga el instalador desde [Releases](https://github.com/agentos/releases) y ejecutalo.

**Windows:**
```
AgentOS_10.0.0_x64-setup.exe
```

El instalador configura todo automaticamente: base de datos local, estructura de carpetas, y servicios en background.

---

## 2. Configurar API Key

Al abrir AgentOS por primera vez, ve a **Settings** (icono de engranaje) y agrega tu API key:

1. Abre **Settings > Proveedores**
2. Ingresa tu **Anthropic API Key** (o OpenAI, Google)
3. Click en **Guardar**

> Tip: Tambien podes usar modelos locales gratis con Ollama. Ve a Settings > LLM Local.

---

## 3. Enviar tu primera tarea

En la pantalla **Home**, escribe una tarea en el campo de texto y presiona Enter:

```
Busca los ultimos 5 emails de trabajo y resume los pendientes
```

El agente ejecuta la tarea, muestra el progreso en tiempo real, y te entrega el resultado.
